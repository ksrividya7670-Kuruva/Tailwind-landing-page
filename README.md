  # Tailwind Hero Landing Page

## Overview

This project is a modern landing page built using HTML and Tailwind CSS. It includes a responsive navigation bar, hero section, information cards, and a floating chat button.

## Features

* Responsive navigation bar
* Hero section with call-to-action buttons
* Task status cards
* Team activity card
* Floating chat button
* Tailwind CSS styling
* Gradient background

## Technologies Used

* HTML5
* Tailwind CSS

## Project Structure

├── index.html
├── style.css (optional)
└── README.md
